odyssey-stencil

A collection of web components based on the Odyssey Design System.

What is this?

This project is a proof of concept that aims to provide web components that provide 1:1 features and functionality with the Odyssey design system.

What needs to be done?

In no particular order and by no means exhaustive:

  • Tests: The only tests today are basic ones to check for rendering. A full pass of tests must be made.
  • Guidelines: Code authoring guidelines as we think about expanding the circle of contributors to a broader group of developers.
  • Accessibility testing: I have questions about shadow-dom, the placement of aria-attributes and how the relationships work across the shadow boundary. Further research and testing is required, specific areas to investigate follow:
    • ID-associated attributes won't work if they have to cross a shadow boundary, like aria-describedby
